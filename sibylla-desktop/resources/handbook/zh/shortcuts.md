# 快捷键

## 通用

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl + P` | 打开命令面板 |
| `Cmd/Ctrl + S` | 保存当前文件 |
| `Cmd/Ctrl + Shift + P` | 打开 AI 模式选择器 |

## 编辑器

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl + B` | 加粗 |
| `Cmd/Ctrl + I` | 斜体 |
| `Cmd/Ctrl + K` | 插入链接 |

## AI 面板

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 发送消息 |
| `Shift + Enter` | 换行 |
| `Escape` | 取消流式响应 |
