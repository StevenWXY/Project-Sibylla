# 通用技能（Common Skills）

本目录包含跨项目可复用的通用开发技能，不特定于 Sibylla 项目。

---

## 技能列表

### 1. [`file-organizer`](file-organizer/SKILL.md)
智能文件和文件夹组织工具，提供上下文理解、重复文件查找、结构优化建议和自动化清理功能。

### 2. [`frontend-design`](frontend-design/SKILL.md)
创建独特的、生产级前端界面，注重高设计质量，避免通用 AI 美学。适用于 web 组件、页面、应用等的创意设计。

### 3. [`skill-creator`](skill-creator/SKILL.md)
创建有效 skills 的完整指南，包含初始化、打包和验证工具。用于扩展 AI 能力的专业知识、工作流或工具集成。

### 4. [`ui-ux-pro-max`](ui-ux-pro-max/SKILL.md)
智能 UI/UX 设计系统生成器，包含 67 种样式、96 种调色板、57 种字体配对、100 条行业特定规则。适用于落地页、仪表板、web 应用等商业产品的标准化设计。

### 5. [`vercel-react-best-practices`](vercel-react-best-practices/SKILL.md)
React 和 Next.js 性能优化指南，包含 45 条规则，涵盖异步操作、打包优化、客户端性能、JavaScript 优化、渲染优化、重渲染控制、服务端优化等 8 个类别。

---

## 使用场景

这些通用技能可在任何项目中使用，为开发团队提供：
- 生产力工具支持
- 前端设计与 UI/UX 系统化方法
- React/Next.js 性能优化最佳实践
- 技能扩展能力
